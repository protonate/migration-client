Welcome to RP Migrator

RP Migrator is a tool to migrate Capture Widget customers to the Widget Server. This is built with Backbone.js, Twitter Bootstrap, MongoDB, and Node.js (with the following requires: express, underscore, knox, and invalidatejs). The code was adapted from the directory-backbone-bootstrap repo by Christophe Coenraets.

Migrate RP

Enter a few characters in the Search Box in the upper right corner of the screen, and select an RP. In the RP view, you can compare load files and Migrate (or Rollback) an RP from using the Widget Server.
